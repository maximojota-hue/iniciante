# YouTube Post Quality Checklist

Use this only when reviewing or improving a YouTube-based Clube 3D Brasil article.

## Must Pass

- The post has a clear angle, not just a summary of the video.
- The first H2 answers the main search question quickly.
- The post includes Brazilian context: price sensitivity, free or cheap STL files, common beginner printers, local marketplaces, groups, or affiliate fit when relevant.
- The article uses at least 3 concrete details from the video: numbers, costs, dates, examples, mistakes, claims, workflow, or warnings.
- Any strong claim from the video is treated carefully. Do not promise easy money, perfect results, or universal compatibility.
- The post adds useful context beyond the transcript.
- There is one section about risk, cost, mistakes, or when not to do it.
- There are 3 to 5 FAQs with direct answers.
- Yoast fields are filled: keyphrase, SEO title, meta description, slug and excerpt.
- Affiliate links are only included when the user provided them.
- Product images open affiliate links in a new tab when used.
- The post is saved as draft unless the user explicitly asks to publish.

## Style

- Brazilian Portuguese.
- Short paragraphs.
- Direct, useful, honest tone.
- No emojis.
- No semicolons.
- No "Key Takeaways" or "Principais aprendizados" section.
- No claims that Clube 3D Brasil tested something unless the user provided proof.

## Free Tool Preference

- Use `yt-dlp` for metadata and transcript extraction.
- Use existing captions before asking for manual transcript.
- Use public video description, chapters, and visible metadata as fallback.
- When working inside chat, do not call external content APIs.
- Use skills and local scripts to avoid repeating long prompts.
